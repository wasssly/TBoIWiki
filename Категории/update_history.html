Patch v1.01

07.11.2014

    Дополнения

  • Добавьте параметр в Options.ini, где вы можете отключить Steam Cloud.
  • Добавьте ярлык ALT + F2 на странице статистики для импорта достижений Steam.


    Исправления

  • Добавлены файлы саундтрека к дополнению Afterbirth.
  • Добавлены некоторые потерянные предметы в пулы предметов.
  • Исправлен сбой при использовании FBO в контекстах OpenGL 2.
  • Исправлены некоторые проблемы с Steam Cloud Sync.
  • Исправлена ошибка, из-за которой Призрак иногда оставался неуязвимым.
  • Исправлены другие ошибки, связанные с игровым процессом.
  • Исправлена проблема с черным экраном в OS X.
  • Исправлена позиция игрока при возвращении с черного рынка.
  • Исправлен вылет при сражении с одним из финальных боссов.
  • Исправлены проблемы с контроллером Xbox 360 в Linux.

Patch v1.02

11.11.2014

    Исправления

  • Исправлены белые текстуры для игроков, использующих OpenGL 2.x из-за более старого графического оборудования. Если используется бета-версия "ogl2", обязательно откажитесь от бета-версии, чтобы правильно обновиться.
  • Исправлено исчезновение окна при запуске на Windows. Игровое окно появлялось на короткий промежуток времени, а затем исчезало, пока музыка все еще играла. Если ошибка все еще присутствует, нажмите Windows + Left или Windows + Right (маленькая клавиша с логотипом Windows и одна из клавиш со стрелками).
  • Исправлен сбой при запуске из-за сломанной options.ini, который можно было устранить путем удаления файла. Теперь игра может обнаруживать и восстанавливать поврежденные файлы .ini.
  • Исправлен сбой при запуске на Linux с ошибками шейдера в логе.
  • Исправлено Божество, ! Достижения «Платиновый бог!», «Настоящий платиновый бог» и «Мертвец» не открываются должным образом.
  • Исправлена проблема с высотой звука музыки во время катсцены «Эпилог», а также некоторые проблемы, связанные с игровым процессом.
  • Исправлен вылет при неработающем звуке.
  • Исправлено сохранение/продолжение во время Boss Rush.
  • Исправлены эксплойты The Lost + Dead Cat.
  • Исправлены эксплойты с бесконечным возрождением предметов.
  • Исправлен эксплойт P2 Trinity Shield.
  • Исправлена ошибка, из-за которой D10 генерировал непобедимых врагов.
